## Requiremenst
- Python 3
- Django


## Usage
### Learning
```sh
$ python train.py
```
### Django
#### Run Server
```sh
$ ./manage.py runserver 0.0.0.0:8080
```
#### Run Test
```sh
$ ./manage.py test
```


## Installation
### Clone this repo
```sh
$ git clone <this repo>
$ cd <this repo>
```
### Create python venv
```sh
# create
$ python -m venv venv
# active
$ source venv/bin/activate
```
### Install python libs
```sh
$ pip install --upgrade pip
$ pip install -r requirements.txt
```
