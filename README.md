# ダジャレ判定エンジン


## Requiremenst
- Python 3
- Django 3.0.3


## Usage
### Learning
```sh
$ python train.py
```
### Django
#### Run Server
```sh
$ ./manage.py runserver 0.0.0.0:8080
```
#### Run Test
```sh
$ ./manage.py test
```


## Installation
### Clone this repo
```sh
$ git clone <this repo>
$ cd <this repo>
```
### Install python libs
```sh
$ pip install -r requirements.txt
```
